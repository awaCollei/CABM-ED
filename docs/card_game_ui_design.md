# 卡牌游戏UI设计说明

## 整体布局

### 屏幕分区
```
┌─────────────────────────────────────┐
│         敌方阵容区域        [投降]  │
│    [敌人1] [敌人2] [敌人3]          │
├─────────────────────────────────────┤
│                            ┌────────┤
│                            │ 回合 X │
│         中央状态提示        │你的回合│
│                            │────────│
│                            │结束回合│
│                            └────────┤
├─────────────────────────────────────┤
│         我方阵容区域                 │
│    [角色1] [角色2] [角色3]          │
├─────────────────────────────────────┤
│┌──────┐                             │
││ 费用 │      手牌区域                │
││ 💎3/3│ [牌1] [牌2] [牌3] [牌4]    │
││牌库:X│                             │
││弃牌:X│                             │
│└──────┘                             │
└─────────────────────────────────────┘
```

### 布局说明

1. **右上角** - 投降按钮
   - 位置：屏幕右上角
   - 尺寸：120x40px
   - 颜色：红色系

2. **右侧中间** - 回合信息面板
   - 位置：屏幕右侧垂直居中
   - 内容：
     - 回合数
     - 当前阶段（你的回合/敌人回合）
     - 结束回合按钮
   - 尺寸：200x160px

3. **左下角** - 费用信息面板
   - 位置：手牌区域左上角
   - 内容：
     - 当前费用/最大费用
     - 牌库数量
     - 弃牌堆数量
   - 尺寸：200x50px

4. **中央** - 状态提示文字
   - 位置：屏幕正中央
   - 显示当前操作提示和战斗信息
   - 带淡入动画效果

## 配色方案

### 主色调
- 背景色：深蓝灰 `#141420` (0.08, 0.08, 0.12)
- 面板背景：深灰蓝 `#1F1F2E` (0.12, 0.12, 0.18)
- 边框色：中灰蓝 `#4D5980` (0.3, 0.35, 0.5)

### 角色卡配色
- 我方角色：
  - 背景：深蓝色 `#264059` (0.15, 0.25, 0.35)
  - 边框：亮蓝色 `#4D80B3` (0.3, 0.5, 0.7)
  - 选中边框：金黄色 `#FFE64D` (1.0, 0.9, 0.3)

- 敌方角色：
  - 背景：深红色 `#592626` (0.35, 0.15, 0.15)
  - 边框：亮红色 `#B34D4D` (0.7, 0.3, 0.3)

- 死亡状态：
  - 背景：深灰 `#333333` (0.2, 0.2, 0.2)
  - 边框：灰色 `#4D4D4D` (0.3, 0.3, 0.3)

### 手牌配色
- 正常状态：
  - 背景：深灰紫 `#2E2E40` (0.18, 0.18, 0.25)
  - 边框：灰色 `#808099` (0.5, 0.5, 0.6)

- 选中状态：
  - 背景：深绿色 `#334D33` (0.2, 0.3, 0.2)
  - 边框：亮绿色 `#4DFF4D` (0.3, 1.0, 0.3)

- 费用不足：
  - 背景：深灰 `#262626` (0.15, 0.15, 0.15)
  - 边框：暗灰 `#4D4D4D` (0.3, 0.3, 0.3)

### 按钮配色
- 结束回合按钮：
  - 正常：深绿 `#338033` (0.2, 0.5, 0.3)
  - 悬停：亮绿 `#4DB366` (0.3, 0.7, 0.4)

- 投降按钮：
  - 正常：深红 `#803333` (0.5, 0.2, 0.2)
  - 悬停：亮红 `#B34D4D` (0.7, 0.3, 0.3)

### 状态颜色
- 生命值：
  - 健康 (>60%)：绿色 `#66FF66` (0.4, 1.0, 0.4)
  - 警告 (30-60%)：橙色 `#FFCC66` (1.0, 0.8, 0.4)
  - 危险 (<30%)：红色 `#FF6666` (1.0, 0.4, 0.4)
  - 死亡：灰色 `#808080` (0.5, 0.5, 0.5)

- 费用：
  - 有费用：蓝色 `#66CCFF` (0.4, 0.8, 1.0)
  - 无费用：红色 `#FF6666` (1.0, 0.4, 0.4)

- Buff效果：
  - 护盾：青色 `#66CCFF` (0.4, 0.8, 1.0)
  - 剧毒：绿色 `#66FF66` (0.4, 1.0, 0.4)

## 视觉元素

### 图标使用
- ❤️ 生命值
- ⚔️ 攻击力
- 💎 费用/能量
- 🛡️ 护盾效果
- 🍃 剧毒效果
- 🎴 卡牌相关

### 字体大小
- 标题：20px
- 区域标签：16px
- 卡牌名称：15px (角色卡), 13px (手牌)
- 属性数值：14px
- Buff状态：12px
- 信息文本：13px

### 圆角和边框
- 面板圆角：12px
- 卡牌圆角：8px
- 按钮圆角：8px
- 边框宽度：2-4px (选中时加粗)

## 交互反馈

### 鼠标状态
- 可点击元素：手型光标 `CURSOR_POINTING_HAND`
- 禁用元素：禁止光标 `CURSOR_FORBIDDEN`
- 默认：箭头光标

### 视觉反馈
1. 选中高亮：边框加粗并变色
2. 悬停效果：按钮颜色变亮
3. 伤害数字：红色数字向上飘动并淡出
4. 状态提示：文字淡入动画

### 卡牌状态
- 可用：正常显示
- 选中：绿色边框高亮
- 费用不足：变灰且禁用
- 死亡：整体变灰

## 间距规范

### 容器间距
- 区域外边距：40px
- 区域内边距：8-15px
- 卡牌间距：20px (角色卡), 10px (手牌)

### 内容间距
- 标签与内容：5px
- 文本行间距：5px
- 区域标题高度：30px

### 面板尺寸
- 费用面板：200x50px (左下角)
- 回合面板：200x160px (右侧中间)
- 投降按钮：120x40px (右上角)
- 结束回合按钮：160x40px (回合面板内)

## 响应式设计

### 最小尺寸
- 角色卡：140x180px
- 手牌：110x150px
- 投降按钮：120x40px
- 结束回合按钮：160x40px
- 费用面板：200x50px
- 回合面板：200x160px

### 布局适配
- 使用锚点和容器自动适配屏幕大小
- 卡牌区域居中对齐
- 状态提示固定在中央
- 费用面板固定在左下角
- 回合面板固定在右侧中间
- 投降按钮固定在右上角

## 动画效果

### 淡入淡出
- 状态提示文字：0.3秒淡入
- 伤害数字：0.8秒向上移动并淡出

### 过渡效果
- 按钮悬停：即时变色
- 卡牌选中：即时边框变化
- 场景切换：0.3-0.5秒淡入淡出

## 可访问性

### 视觉辅助
- 使用图标+文字组合
- 颜色编码配合文字说明
- 清晰的状态区分

### 交互提示
- 悬停提示显示卡牌描述
- 光标变化指示可交互性
- 禁用状态明确标识
