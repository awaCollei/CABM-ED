# 卡牌游戏机制说明

## 游戏流程

### 1. 准备阶段
- 玩家选择3个角色卡进入战斗
- 对手为3个"普通敌人"角色卡
- 初始牌库包含所有手牌类型，每种3张

### 2. 战斗阶段

#### 回合制
- 玩家回合 → 敌人回合 → 循环
- 每回合开始时：
  - 恢复费用到最大值（3点）
  - 抽取2张牌
  - 处理buff效果（如剧毒）

#### 玩家回合操作
1. 使用手牌（消耗费用）
2. 结束回合

### 3. 核心机制

#### 费用系统
- 每回合开始时恢复到3点费用
- 使用手牌需要消耗对应费用
- 费用不足时无法使用手牌

#### 抽牌机制
- 每回合开始抽2张牌
- 手牌上限10张
- 牌库用完时，弃牌堆洗回牌库

#### 角色卡属性
- HP（生命值）：角色的生命值，降到0时阵亡
- ATK（攻击力）：攻击时造成的伤害
- 技能：特殊能力（当前版本未实现）

#### 手牌类型

1. **攻击** (2费用)
   - 选择己方角色，再选择敌方目标进行攻击
   - 造成等于角色攻击力的伤害

2. **铁壁防御** (2费用)
   - 为己方角色添加护盾
   - 下次受到的伤害降低2点

3. **治愈之光** (3费用)
   - 为己方角色恢复2点生命值
   - 不能超过最大生命值

4. **剧毒之刃** (2费用)
   - 为敌方角色施加剧毒效果
   - 持续3回合，每回合开始时受到1点伤害

5. **命运之轮** (1费用)
   - 抽取3张牌

### 4. 胜负判定

#### 胜利条件
- 所有敌方角色阵亡

#### 失败条件
- 所有己方角色阵亡
- 玩家投降

## 游戏策略提示

1. 合理分配费用，不要浪费
2. 及时使用治疗和护盾保护关键角色
3. 利用剧毒持续消耗敌人
4. 命运之轮可以快速补充手牌
5. 注意手牌上限，避免浪费抽牌机会
