# 助眠模式新功能说明

## 后台模式开关

### 功能描述
- 添加了"后台"开关按钮，位于界面左上角"动态/静态"按钮旁边
- 按钮显示当前状态："后台:开" 或 "后台:关"

### 平台支持
- **安卓平台**：开启后台模式时，会设置屏幕常亮（`DisplayServer.screen_set_keep_on(true)`），允许在熄屏后继续进行任务
- **其他平台**：按钮可见但功能仅在安卓平台生效

### 配置保存
- 后台模式状态会保存到 `user://sleep_config.json`
- 下次进入助眠模式时会自动恢复上次的设置
- 退出助眠模式时会自动关闭屏幕常亮功能

## 状态栏展开/收起状态保存

### 功能描述
- 状态栏的打开/收起状态现在会被保存
- 点击状态栏的 ▲/▼ 按钮可以展开或收起状态栏
- 状态会立即保存到配置文件

### 配置保存
- 状态栏展开状态保存到 `user://sleep_config.json` 的 `is_status_bar_expanded` 字段
- 下次进入助眠模式时会自动恢复上次的展开/收起状态

## 配置文件结构

```json
{
	"is_dynamic_background": true,
	"is_background_mode_enabled": false,
	"is_status_bar_expanded": true,
	"bg_volume": -10.0,
	"voice_volume": 0.0,
	"trigger_volume": 0.0
}
```

## 使用说明

1. **开启后台模式**（安卓）：
   - 点击左上角"后台:关"按钮
   - 按钮变为"后台:开"
   - 屏幕将保持常亮，即使锁屏也会继续播放

2. **关闭后台模式**：
   - 点击"后台:开"按钮
   - 按钮变为"后台:关"
   - 屏幕恢复正常行为

3. **状态栏控制**：
   - 点击状态栏的 ▲ 按钮收起状态栏
   - 点击 ▼ 按钮展开状态栏
   - 状态会自动保存

## 技术实现

- 使用 `DisplayServer.screen_set_keep_on()` API 控制屏幕常亮
- 所有设置通过 JSON 格式保存到用户数据目录
- 退出时自动清理资源并关闭屏幕常亮
